# Gemini AI Chatbot - PHP 🤖

Este es un proyecto backend desarrollado en PHP que implementa un chatbot interactivo conectado directamente con la API de Gemini (Google AI). Permite procesar consultas de lenguaje natural y recibir respuestas automatizadas basadas en modelos avanzados de inteligencia artificial.

## 🚀 Características
- Integración directa con la API de Gemini.
- Gestión segura de credenciales mediante variables de entorno (`.env`).
- Arquitectura modular y limpia con Composer para la gestión de dependencias.

## 🛠️ Tecnologías y Herramientas utilizadas
- **Lenguaje:** PHP 8.x
- **Gestor de Dependencias:** Composer
- **Variables de Entorno:** vlucas/phpdotenv (o la librería que uses para leer el .env)
- **Modelos de IA:** Gemini API

## 📋 Estructura del Proyecto
- `src/`: Contiene la lógica central del chatbot y los servicios de la API.
- `public/`: Punto de entrada de la aplicación.
- `bootstrap.php`: Configuración inicial y carga de variables de entorno.
